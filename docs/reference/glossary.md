# 용어집

PromptBlocks와 프롬프트 엔지니어링 관련 용어를 정리합니다.

## A-C

### AI 이미지 생성 (AI Image Generation)

**영어**: AI Image Generation  
**정의**: 인공지능이 텍스트 프롬프트를 기반으로 이미지를 생성하는 기술  
**PromptBlocks에서**: Decompose와 Assemble을 통해 프롬프트를 최적화하여 더 나은 결과를 얻을 수 있습니다.

### Appearance (인물 외형)

**한국어**: 인물 외형  
**정의**: 인물의 외모적 특징을 설명하는 블록 타입  
**예시**: "long black hair", "blue eyes", "freckles"

### Assemble (조립)

**한국어**: 조립  
**정의**: 저장된 블록을 조합하여 완성된 프롬프트를 만드는 기능  
**PromptBlocks에서**: Library의 블록을 선택하고 순서를 조정하여 프롬프트 생성

## B-D

### Batch Decomposition (배치 분해)

**한국어**: 배치 분해  
**정의**: 여러 이미지를 한 번에 분석하는 기능  
**PromptBlocks에서**: 최대 5개 이미지를 동시에 분석

### Block (블록)

**한국어**: 블록  
**정의**: 프롬프트의 의미 단위로 분해된 요소  
**PromptBlocks에서**: 13가지 타입으로 분류되어 관리됨

### Collection (컬렉션)

**한국어**: 컬렉션  
**정의**: 블록을 그룹화하여 관리하는 단위  
**PromptBlocks에서**: 프로젝트별, 주제별로 블록을 묶어서 정리

### Decompose (분해)

**한국어**: 분해  
**정의**: 이미지나 텍스트를 13개 블록으로 분석하는 기능  
**PromptBlocks에서**: AI가 자동으로 블록 단위로 분해

## F-P

### Face Consistency (얼굴 일관성)

**한국어**: 얼굴 일관성  
**정의**: 같은 캐릭터의 외형을 유지하면서 다른 상황의 이미지를 생성하는 기능  
**PromptBlocks에서**: 얼굴 앵커 저장과 블록 잠금으로 구현

### Negative Prompt (네거티브 프롬프트)

**한국어**: 부정 프롬프트  
**정의**: 이미지에서 제외하고 싶은 요소를 지정하는 프롬프트  
**예시**: "--no text, watermark, blurry"

### Preset (프리셋)

**한국어**: 프리셋  
**정의**: 자주 사용하는 블록 조합을 저장한 템플릿  
**PromptBlocks에서**: Assemble에서 저장하고 불러오기 가능

### Prompt Engineering (프롬프트 엔지니어링)

**한국어**: 프롬프트 엔지니어링  
**정의**: AI에게 원하는 결과를 얻기 위해 프롬프트를 체계적으로 설계하는 기술  
**PromptBlocks에서**: 블록 단위로 프롬프트를 관리하여 재사용성과 일관성 확보

## R-S

### Randomize (무작위 조합)

**한국어**: 무작위 조합  
**정의**: 선택된 블록을 랜덤하게 교체하는 기능  
**PromptBlocks에서**: 잠금되지 않은 블록만 대상으로 교체

### SREF (Style Reference)

**한국어**: 스타일 참조  
**정의**: Midjourney에서 특정 이미지의 스타일을 참조하는 파라미터  
**사용법**: `--sref [코드]`

### Style Weight (스타일 웨이트)

**한국어**: 스타일 가중치  
**정의**: SREF 스타일의 강도를 조절하는 파라미터  
**사용법**: `--sw 0-1000`

## T-Z

### Tag (태그)

**한국어**: 태그  
**정의**: 블록에 부여되는 키워드로 검색과 필터링에 사용  
**PromptBlocks에서**: AI가 자동 추출 또는 사용자가 직접 추가

### Template (템플릿)

**한국어**: 템플릿  
**정의**: 미리 정의된 블록 구조  
**PromptBlocks에서**: 초보자를 위한 기본 블록 조합 제공

### Usage Limit (사용량 제한)

**한국어**: 사용량 제한  
**정의**: 무료/유료 플랜에 따른 분해 횟수 제한  
**PromptBlocks에서**: 무료 플랜은 월 20회, Pro는 무제한

## 자주 쓰는 영어 용어

| 영어         | 한국어     | 설명                               |
| ------------ | ---------- | ---------------------------------- |
| Prompt       | 프롬프트   | AI에게 주는 명령/설명 텍스트       |
| Seed         | 시드값     | 재현 가능한 결과를 위한 랜덤값     |
| Upscaling    | 업스케일링 | 이미지 해상도 향상                 |
| Inference    | 추론       | AI가 이미지를 생성하는 과정        |
| Latent Space | 잠재 공간  | AI가 이미지를 표현하는 수학적 공간 |
| Token        | 토큰       | AI가 처리하는 텍스트 단위          |
