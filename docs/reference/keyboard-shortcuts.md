# 키보드 단축키

PromptBlocks의 모든 키보드 단축키 목록입니다.

## 단축키 목록

| 단축키        | Mac           | Windows/Linux    | 기능                      | 페이지    |
| ------------- | ------------- | ---------------- | ------------------------- | --------- |
| 분해 실행     | ⌘ + D         | Ctrl + D         | 이미지/텍스트 분해 시작   | Decompose |
| 블록 저장     | ⌘ + S         | Ctrl + S         | 선택한 블록 저장          | Decompose |
| 프롬프트 복사 | ⌘ + C         | Ctrl + C         | 생성된 프롬프트 복사      | Assemble  |
| 모달 닫기     | Esc           | Esc              | 열린 모달/다이얼로그 닫기 | 전체      |
| 도움말 표시   | ?             | ?                | 단축키 도움말 모달 열기   | 전체      |
| 검색 포커스   | /             | /                | 검색창으로 포커스 이동    | Library   |
| 새 컬렉션     | ⌘ + Shift + N | Ctrl + Shift + N | 새 컬렉션 생성            | Library   |
| 무작위 조합   | ⌘ + R         | Ctrl + R         | 블록 무작위 교체          | Assemble  |
| 프리셋 저장   | ⌘ + P         | Ctrl + P         | 현재 조합 프리셋으로 저장 | Assemble  |

---

## 상세 설명

### ⌘/Ctrl + D: 분해 실행

**Decompose 페이지에서**

- 이미지/텍스트 분석을 즉시 시작합니다.
- API 키가 설정되어 있어야 합니다.
- 입력란에 포커스가 있을 때는 작동하지 않습니다.

### ⌘/Ctrl + S: 블록 저장

**Decompose 페이지에서**

- 선택된 블록들을 라이브러리에 저장합니다.
- 저장 전 컬렉션 선택 모달이 열립니다.

### ⌘/Ctrl + C: 프롬프트 복사

**Assemble 페이지에서**

- 완성된 프롬프트를 클립보드에 복사합니다.
- 텍스트 선택 상태에서는 일반 복사 동작을 합니다.

### Esc: 모달 닫기

**전체 페이지**

- 열린 모달, 다이얼로그, 드롭다운을 닫습니다.
- 모달이 없을 때는 아무 동작도 하지 않습니다.

### ?: 도움말 표시

**전체 페이지**

- 단축키 도움말 모달을 엽니다.
- 입력란에 포커스가 없을 때 작동합니다.

---

## 단축키 사용 팁

### ✅ 효율적인 워크플로우

1. **Decompose**에서 ⌘+D로 빠르게 분석
2. 블록 선택 후 ⌘+S로 저장
3. **Assemble**에서 ⌘+C로 복사
4. Midjourney에 붙여넣기

### ⚠️ 주의사항

- **텍스트 입력 중**: 입력란에 포커스가 있을 때는 단축키가 비활성화됩니다.
- **브라우저 충돌**: 일부 단축키는 브라우저 기본 단축키와 충돌할 수 있습니다.
- **Mac/Windows 차이**: Mac에서는 ⌘(Command), Windows/Linux에서는 Ctrl를 사용합니다.

### 🎨 커스텀 단축키

현재는 커스텀 단축키를 지원하지 않습니다. 향후 업데이트에서 고려될 예정입니다.
