# 블록 라이브러리 관리하기

라이브러리에서 블록을 효율적으로 검색, 필터링, 정리하는 방법을 알아봅니다.

## 개요

**Library**는 Decompose에서 저장한 블록을 관리하는 공간입니다. 태그, 컬렉션, 즐겨찾기를 활용하여 블록을 체계적으로 정리할 수 있습니다.

## 전제조건

- 저장된 블록이 최소 1개 이상 있어야 함
- (선택사항) 컬렉션 생성

## 단계별 가이드

### 1. Library 페이지로 이동

좌측 납비게이션에서 **"Library"**를 클릭하세요.

### 2. 블록 검색

**검색창**에 키워드를 입력하세요:

- 블록 내용 검색
- 태그 검색
- 타입 검색 (예: "스타일", "조명")

> 💡 **팁**: 검색어는 실시간으로 필터링됩니다 (300ms 디바운스)

### 3. 태그로 필터링

**태그 클라우드**에서 원하는 태그를 클릭:

1. 인기 태그 목록 확인
2. 원하는 태그 클릭
3. 해당 태그가 포함된 블록만 표시
4. 여러 태그 선택 가능 (AND/OR 모드)

![라이브러리 페이지](../images/app-library.png)
*블록 라이브러리에서 저장된 블록들을 관리합니다*

### 4. 컬렉션별 보기

**컬렉션 필터** 드롭다운에서 선택:

- "전체 컬렉션" - 모든 블록
- "미분류" - 컬렉션에 속하지 않은 블록
- 특정 컬렉션 - 해당 컬렉션의 블록만

### 5. 정렬 변경

**정렬 옵션**:

- **최신순**: 가장 최근에 추가/수정된 순
- **즐겨찾기 우선**: 즐겨찾기한 블록을 위로
- **타입순**: 블록 타입별 알파벳 순

### 6. 즐겨찾기 관리

**즐겨찾기 추가/제거**:

1. 블록 우측의 별 아이콘 클릭
2. 채워진 별: 즐겨찾기됨
3. 빈 별: 즐겨찾기 안됨

**즐겨찾기만 보기**:

- 상단의 "⭐ 즐겨찾기만" 버튼 클릭

### 7. 블록 복사

블록 내용을 클립보드에 복사:

1. 블록 우측의 복사 아이콘 클릭
2. "복사되었습니다!" 토스트 확인
3. 원하는 곳에 Ctrl+V로 붙여넣기

### 8. 블록 삭제

**단일 삭제**:

1. 블록 우측의 휴지통 아이콘 클릭
2. 삭제 확인 모달에서 "삭제" 클릭

> ⚠️ **주의**: 삭제된 블록은 복구할 수 없습니다!

## 고급 기능

### 아코디언 펼치기/접기

**모두 펼치기**: 모든 블록 타입 섹션 펼침
**모두 접기**: 모든 섹션 접음
**개별 제어**: 각 타입 헤더 클릭으로 개별 펼침/접음

### 블록 타입별 통계

각 블록 타입 헤더에 저장된 블록 수가 표시됩니다:

- "주제 유형 (15개)"
- "스타일 (8개)"

## 팁과 모범사례

### ✅ 효과적인 검색

- **짧은 키워드**: "woman"보다 "Korean woman"
- **태그 활용**: 자주 쓰는 태그 기억해두기
- **복합 검색**: 검색어 + 컬렉션 + 태그 조합

### ✅ 라이브러리 관리

- **정기 정리**: 사용하지 않는 블록 삭제 (월 1회)
- **의미 있는 컬렉션명**: "Temp" 대신 "2024 봄 캠페인"
- **일관된 태그**: "portrait", "portraits" 중 하나로 통일

### ⚠️ 주의사항

- **삭제 전 확인**: 중요한 블록은 삭제 전 백업
- **컬렉션 정리**: 너무 많은 컬렉션은 오히려 혼란 야기
- **태그 남용**: 너무 많은 태그는 검색 효율 저하
