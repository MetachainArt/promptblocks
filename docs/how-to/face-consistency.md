# 얼굴 일관성 유지하기

같은 캐릭터를 다양한 상황에서 일관되게 생성하는 방법을 알아봅니다.

## 개요

얼굴 일관성(Face Consistency) 기능은 캐릭터의 외형을 고정하고 배경, 의상, 포즈 등만 변경하여 같은 인물이 등장하는 다양한 이미지를 생성할 수 있게 합니다.

## 전제조건

- 기준이 될 캐릭터 이미지
- Decompose에서 얼굴 관련 블록 추출 완료

## 단계별 가이드

### 1. 캐릭터 이미지 분석

1. **Decompose** 페이지로 이동
2. 캐릭터 이미지 업로드
3. **"이미지 분석 시작"** 클릭
4. 특히 **Subject Type**과 **Appearance** 블록 확인

### 2. 얼굴 기준 저장

1. 분석 결과 하단의 **"얼굴 기준 저장"** 버튼 클릭
2. 또는 개별적으로 블록 저장

> 💡 **팁**: "Korean woman, long black hair, brown eyes" 같은 구체적 설명이 효과적입니다.

### 3. 설정 페이지 확인

1. **설정** 페이지로 이동
2. **얼굴 일관성** 섹션 확인
3. 저장된 얼굴 앵커와 참조 이미지 확인
4. **자동 적용** 옵션 켜기

### 4. Assemble에서 블록 잠금

1. **Assemble** 페이지로 이동
2. 얼굴 관련 블록 추가 (**Subject Type**, **Appearance**)
3. 해당 블록의 **잠금 아이콘** 클릭 (🔒)
4. 얼굴 앵커가 자동으로 프롬프트에 추가되는지 확인

### 5. 다른 요소 변경

잠금되지 않은 블록들을 자유롭게 변경:

- **Background**: 다양한 장소
- **Outfit**: 다른 의상
- **Pose/Expression**: 다양한 포즈와 표정
- **Lighting**: 다른 조명 조건

### 6. 무작위 조합 활용

1. **"무작위"** 버튼 클릭
2. 잠금된 얼굴 블록은 유지됨
3. 나머지 블록만 변경됨
4. 같은 캐릭터, 다른 상황의 이미지 생성

## 팁과 모범사례

### ✅ 효과적인 얼굴 앵커 작성

- **구체적**: "Asian woman"보다 "Korean woman in her 20s"
- **고유 특징**: "mole on left cheek", "freckles across nose"
- **헤어스타일**: "bob cut with bangs", "waist-length wavy hair"

### ✅ 일관성 유지 팁

- **참조 이미지**: 여러 각도의 캐릭터 이미지 참조로 활용
- **스타일 일관성**: 같은 스타일 아티스트 유지
- **조명 통일**: 비슷한 조명 조건으로 연속성 강조

### ⚠️ 주의사항

- **과도한 잠금**: 너무 많은 블록을 잠금하면 창의성 저하
- **앵커 충돌**: 다른 캐릭터 이미지 분석 시 기존 앵커 덮어씀
- **한계**: 100% 동일한 캐릭터는 AI의 한계로 어려움
