# 이미지에서 프롬프트 추출하기

이미지를 업로드하여 AI가 자동으로 13개 요소로 분해하는 방법을 알아봅니다.

## 개요

PromptBlocks의 **Decompose** 기능은 이미지를 분석하여 AI가 사용한 것으로 추정되는 프롬프트를 13개 블록으로 추출합니다.

## 전제조건

- PromptBlocks 계정
- 분석할 이미지 (JPG, PNG, WebP, 최대 20MB)
- OpenAI 또는 Gemini API 키

## 단계별 가이드

### 1. Decompose 페이지로 이동

좌측 납비게이션에서 **"Decompose"**를 클릭하세요.

### 2. 이미지 업로드

**방법 A: 드래그앤드롭**

1. 컴퓨터에서 이미지 파일을 선택
2. "이미지를 끌어다 놓으세요" 영역으로 드래그
3. 파일을 놓으면 업로드 완료

**방법 B: 파일 선택**

1. 업로드 영역을 클릭
2. 파일 선택 다이얼로그에서 이미지 선택
3. 열기 버튼 클릭

> 💡 **팁**: 선명하고 복잡하지 않은 이미지일수록 분석 정확도가 높아집니다.

### 3. AI 모델 선택

**모델 선택 드롭다운**에서 원하는 AI를 선택하세요:

| 모델           | 특징             | 추천 상황            |
| -------------- | ---------------- | -------------------- |
| GPT-5.2        | 가장 정확한 분석 | 일반적인 이미지      |
| GPT-5-mini     | 빠른 처리        | 간단한 이미지        |
| Gemini 3 Pro   | 한국어에 강함    | 한국인/아시아 캐릭터 |
| Gemini 3 Flash | 빠른 처리        | 프로토타이핑         |

### 4. 분석 실행

**"이미지 분석 시작"** 버튼을 클릭하세요.

분석에는 10초에서 2분 정도 소요됩니다. 진행 상태가 화면에 표시됩니다.

### 5. 결과 확인

분석이 완료되면 13개 블록이 표시됩니다:

- ✅ **성공적으로 추출된 블록**: 내용이 채워져 있음
- ⚪ **추출되지 않은 블록**: 이미지에 해당 요소가 없거나 확인 불가

[스크린샷: 분석 결과 화면]

### 6. 블록 편집 (선택사항)

각 블록의 내용을 클릭하여 직접 수정할 수 있습니다:

1. 수정할 블록의 텍스트 클릭
2. 내용 편집
3. Enter 키 또는 화면 다른 곳 클릭으로 저장

> ⚠️ **주의**: 블록 내용을 수정하면 라이브러리에 저장될 때 수정된 내용이 저장됩니다.

### 7. 블록 선택 및 저장

1. 저장할 블록의 체크박스를 선택
2. 하단의 **"선택한 블록 저장"** 버튼 클릭
3. 저장할 컬렉션 선택 (또는 새 컬렉션 생성)

[스크린샷: 블록 저장 화면]

## 배치 분석 (여러 이미지)

최대 5개의 이미지를 동시에 분석할 수 있습니다:

1. 이미지 선택 시 여러 파일 선택
2. 또는 이미지를 하나씩 추가
3. **"배치 분석 시작"** 버튼 클릭
4. 각 이미지의 진행 상태가 개별적으로 표시됨
5. 모든 분석 완료 후 일괄 저장 가능

## 팁과 모범사례

### ✅ 효과적인 이미지 선택

- **고해상도**: 1024x1024 이상 권장
- **선명한 초점**: 흐릿한 이미지는 분석 정확도 저하
- **단순한 구도**: 너무 복잡한 이미지는 오분석 가능

### ✅ 블록 활용법

- **재사용 가능한 블록만 저장**: 한 번만 쓸 내용은 저장 불필요
- **컬렉션으로 정리**: 프로젝트별/주제별로 분류
- **즐겨찾기 활용**: 자주 쓰는 블록은 별표 표시

### ⚠️ 주의사항

- **저작권**: 타인의 이미지 분석 시 저작권 주의
- **개인정보**: 얼굴이 포함된 이미지는 개인정보 보호 고려
- **정확도 한계**: AI 분석은 추정치이므로 100% 정확하지 않을 수 있음

## 문제 해결

| 문제           | 원인        | 해결책                      |
| -------------- | ----------- | --------------------------- |
| 분석 실패      | API 키 오류 | 설정 페이지에서 API 키 확인 |
| 결과 부정확    | 이미지 품질 | 더 선명한 이미지 사용       |
| 일부 블록 누락 | 이미지 특성 | 해당 요소가 없을 수 있음    |
