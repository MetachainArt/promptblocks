# 오류 메시지 해결

PromptBlocks 사용 중 발생할 수 있는 오류와 해결 방법입니다.

## API 오류

### "API 키가 설정되지 않았습니다"

**원인**:

- OpenAI 또는 Gemini API 키가 입력되지 않음
- API 키가 잘못된 형식

**해결책**:

1. 좌측 메뉴에서 **설정** 클릭
2. **API 키** 섹션으로 이동
3. 유효한 API 키 입력
4. **저장** 버튼 클릭

> 💡 **팁**: API 키는 [OpenAI](https://platform.openai.com) 또는 [Google AI Studio](https://aistudio.google.com)에서 발급받을 수 있습니다.

---

### "API 호출 한도를 초과했습니다"

**원인**:

- 무례 플랜 월 20회 분해 한도 초과
- AI 제공업체의 Rate Limit 초과

**해결책**:

1. **사용량 확인**: Analytics 페이지에서 현재 사용량 확인
2. **Pro 플랜 업그레이드**: 설정 > 요금제에서 업그레이드
3. **잠시 대기**: Rate Limit의 경우 몇 분 후 재시도

---

### "잘못된 API 응답"

**원인**:

- AI 서버 일시적 오류
- 네트워크 연결 문제
- 이미지 형식 오류

**해결책**:

1. **재시도**: "재시도" 버튼 클릭
2. **이미지 확인**: 다른 이미지로 테스트
3. **다른 모델**: GPT 대신 Gemini 등 다른 모델 시도
4. **새로고침**: 페이지 새로고침 후 다시 시도

---

## 업로드 오류

### "파일 크기가 너무 큽니다"

**원인**:

- 이미지 파일이 20MB 초과

**해결책**:

1. 이미지 편집 프로그램에서 크기 조정
2. [TinyPNG](https://tinypng.com) 등으로 압축
3. JPG/PNG 형식으로 변환

---

### "지원하지 않는 이미지 형식입니다"

**원인**:

- BMP, TIFF, RAW 등 비지원 형식
- 파일 확장자 변경된 이미지

**해결책**:

1. JPG, PNG, WebP 형식으로 변환
2. 이미지 뷰어에서 "다른 이름으로 저장" 사용
3. Online-Convert 등의 변환 도구 활용

---

### "업로드 실패"

**원인**:

- 네트워크 연결 불안정
- 서버 일시적 오류
- 브라우저 캐시 문제

**해결책**:

1. **네트워크 확인**: 인터넷 연결 상태 확인
2. **캐시 삭제**: 브라우저 캐시 및 쿠키 삭제
3. **다른 브라우저**: Chrome, Firefox 등 다른 브라우저 시도
4. **시간 경과 후 재시도**: 서버 문제일 수 있음

---

## 처리 오류

### "분석 실패"

**원인**:

- 이미지 품질이 너무 낮음
- 이미지가 너무 복잡하거나 추상적
- AI가 인식할 수 없는 콘텐츠

**해결책**:

1. **더 선명한 이미지**: 해상도가 높은 이미지 사용
2. **간단한 이미지**: 복잡하지 않은 구도 선택
3. **텍스트 모드**: 이미지 대신 텍스트로 직접 입력
4. **다른 모델**: 다른 AI 모델로 시도

---

### "블록 저장 실패"

**원인**:

- 네트워크 연결 문제
- 블록 내용이 비어있음
- Supabase 연결 오류

**해결책**:

1. **내용 확인**: 블록 내용이 비어있지 않은지 확인
2. **재시도**: 저장 버튼 다시 클릭
3. **로컬 모드**: 비로그인 상태에서 localStorage 사용
4. **로그아웃/로그인**: 세션 문제일 수 있음

---

### "네트워크 오류"

**원인**:

- 인터넷 연결 끊김
- 방화벽/프록시 설정
- DNS 문제

**해결책**:

1. **연결 확인**: 다른 웹사이트 접속 테스트
2. **VPN 해제**: VPN 사용 중이라면 일시 중지
3. **방화벽 확인**: 기업 네트워크의 경우 IT 담당자 문의
4. **DNS 변경**: Google DNS(8.8.8.8) 사용

---

## 브라우저/성능 문제

### "페이지가 느리게 로딩됩니다"

**원인**:

- 저장된 블록이 너무 많음 (500개 이상)
- 브라우저 확장 프로그램 충돌
- 하드웨어 가속 비활성화

**해결책**:

1. **블록 정리**: 사용하지 않는 블록 삭제
2. **확장 프로그램 비활성화**: 광고 차단기 등 일시 중지
3. **하드웨어 가속 활성화**: Chrome 설정 > 시스템 > 하드웨어 가속
4. **캐시 삭제**: 브라우저 데이터 정리

---

### "localStorage가 가득 찼습니다"

**원인**:

- 비로그인 상태에서 너무 많은 블록 저장
- 브라우저 저장 용량 한도(5-10MB)

**해결책**:

1. **로그인**: 계정에 로그인하여 클라우드 저장
2. **블록 정리**: 불필요한 블록 삭제
3. **데이터 낭비기**: 설정 > 데이터 낭비기로 JSON 백업 후 삭제

---

### "화면이 깨져 보입니다"

**원인**:

- CSS 캐시 문제
- 브라우저 호환성 문제
- 확장 프로그램 간섭

**해결책**:

1. **강력 새로고침**: Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)
2. **다른 브라우저**: Chrome, Safari, Firefox 중 하나로 시도
3. **시크릿 모드**: 확장 프로그램 없는 시크릿 모드로 테스트

---

## 계정/인증 오류

### "로그인 세션이 만료되었습니다"

**원인**:

- 장시간 비활성
- 보안 상의 이유로 세션 종료

**해결책**:

1. **다시 로그인**: 로그아웃 후 다시 로그인
2. **비밀번호 재설정**: 비밀번호를 잊은 경우 재설정

---

### "이메일 인증이 필요합니다"

**원인**:

- 회원가입 후 이메일 인증 미완료

**해결책**:

1. **이메일 확인**: 가입한 이메일의 받은편지함/스팸함 확인
2. **인증 링크 클릭**: 이메일의 인증 링크 클릭
3. **재발송**: 인증 메일 재발송 요청

---

## 일반 문제 해결 체크리스트

문제가 지속될 때 시도해 볼 것:

- [ ] 브라우저 새로고침 (F5)
- [ ] 강력 새로고침 (Ctrl/Cmd + Shift + R)
- [ ] 다른 브라우저에서 테스트
- [ ] 시크릿/시크릿 모드로 테스트
- [ ] 브라우저 캐시 및 쿠키 삭제
- [ ] VPN/프록시 비활성화
- [ ] 방화벽 설정 확인
- [ ] 다른 네트워크에서 테스트 (모바일 핫스팟 등)
