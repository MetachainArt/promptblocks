# 5분 퀵스타트

> 🚀 5분 만에 PromptBlocks로 첫 프롬프트 만들기

## 1분 개요

**PromptBlocks**는 AI 이미지 생성 프롬프트를 분해하고, 저장하고, 재조합하는 도구입니다. Pinterest의 멋진 이미지를 분석하거나, 내 프롬프트를 체계적으로 관리하고 싶을 때 사용하세요.

**이런 분에게 추천해요**:

- Midjourney, DALL-E, Stable Diffusion 사용자
- 프롬프트를 체계적으로 정리하고 싶은 분
- 일관된 캐릭터/스타일 이미지를 만들고 싶은 분

---

## 5분만에 시작하기

### 1단계: 로그인 (30초)

[사이트 주소]에 접속해서 로그인하거나 새 계정을 만드세요.

### 2단계: Decompose로 이동 (10초)

좌측 메뉴에서 **"Decompose"**를 클릭하세요.

### 3단계: 이미지 업로드 (1분)

1. "이미지를 끌어다 놓으세요" 영역에 분석하고 싶은 이미지를 드래그
2. 또는 클릭해서 파일 선택
3. **최대 20MB, JPG/PNG/WebP** 형식 지원

[스크린샷: 이미지 업로드 영역]

### 4단계: AI 분석 실행 (1-2분)

1. AI 모델 선택 (GPT-5.2 또는 Gemini 3 Pro 추천)
2. **"이미지 분석 시작"** 버튼 클릭
3. AI가 이미지를 13개 요소로 분해하는 동안 대기

[스크린샷: 분석 버튼]

### 5단계: 결과 확인 및 저장 (1분)

분석이 완료되면 13개 블록이 표시됩니다:

- ✅ 마음에 드는 블록을 선택
- 📝 필요하면 블록 내용 직접 수정
- 💾 **"선택한 블록 저장"** 버튼 클릭

[스크린샷: 분석 결과 화면]

### 6단계: Assemble로 이동 (10초)

좌측 메뉴에서 **"Assemble"**를 클릭하세요.

### 7단계: 블록 조립 (1분)

1. **"블록 추가"** 버튼 클릭
2. 라이브러리에서 원하는 블록 선택
3. 블록 순서 변경 (드래그앤드롭)
4. 작가 스타일 추가 (원하는 경우)

[스크린샷: 블록 조립 화면]

### 8단계: 프롬프트 복사 (10초)

완성된 프롬프트 하단의 **"복사"** 버튼을 클릭하고, Midjourney나 DALL-E에 붙여넣으세요!

---

## 핵심 개념 이해하기

```
이미지/텍스트 ──► [Decompose] ──► 13개 블록 ──► [Library] ──► 저장/관리
                                               │
                                               ▼
완성된 프롬프트 ◄── [Assemble] ◄── 블록 선택
```

### 🔬 Decompose (분해)

이미지나 텍스트를 AI가 분석해서 13개 요소(블록)로 쪼갭니다.

### 📚 Library (라이브러리)

분핸된 블록을 저장하고 태그, 컬렉션으로 관리합니다.

### 🧩 Assemble (조립)

저장된 블록을 조합해서 새로운 프롬프트를 만듭니다.
